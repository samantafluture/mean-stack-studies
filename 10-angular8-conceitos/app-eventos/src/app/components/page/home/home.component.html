Projeto desenvolvido com ♥ por Samanta Fluture

Tecnologias

  • Front-end: Angular
  • Web Service: Node.js com Express
  • Banco de dados: MongoDB com Mongoose
  • APIs externas: GitHub API e Quotable API
  • UI: MDbootstrap e Font Awesome

Instalação

  1. Baixe o projeto no
  2. Não esqueça de dar uma estrela ★
  3. Tenha instalado: Node.js, npm e nodemon
  4. Rode no terminal os comandos abaixo

Na pasta do projeto , rode: 

    npm install
    npm start

Na pasta do projeto , rode: 

    npm install
    nodemoon app

Acesse localhost:4200 e divirta-se!
